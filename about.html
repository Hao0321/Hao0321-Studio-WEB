為什麼做遊戲大廳？

「Hao0321 遊戲大廳」（game.hao0321.com）是工作室的免費娛樂產品線，也是技術練功場。每款小遊戲都是我親手從零寫出來的 — React + Canvas/HTML，沒有引擎、沒有模板、沒有外包。

目的有三：

  • 練技術 — 從物理引擎（救救狗狗的畫線模擬）、AI 棋牌 AI（Chess Master、狼人殺）、網頁即時戰鬥（HAO SURVIVOR、DODGE MASTER），到經濟模擬（台灣大富翁），每款都是一次把理論落地的機會。
  • 做能分享的作品 — 比起作品集裡的靜態截圖，能讓朋友點開就玩的遊戲更有說服力。
  • 持續收到回饋 — 排行榜、簽到、成就系統不是為了黏著度，是為了知道哪些設計真的有趣、哪些計分不公平。

14 款遊戲涵蓋策略、動作、益智、卡牌、桌遊五大類型，從單人魂魄派 Roguelike 到多人本地對戰都有。全站免費、不需下載、不需註冊也能玩。登入（用 Google）純粹是為了存分數。

技術堆疊

目前整個工作室的網站與後端，完全跑在一個人可負擔的雲端配置上，月固定成本不到新台幣 200 元：

FrontendHTML + Vanilla JS + React
HostingCloudflare Pages
BackendCloudflare Workers
DatabaseCloudflare D1 (SQLite)
AuthGoogle OAuth + JWT
CDN / SSLCloudflare
3D / VFXHoudini, Blender, Nuke
Game EnginesUnreal, Unity

選擇 Cloudflare 堆疊的理由很簡單：全球邊緣運算的延遲極低（API 回應 < 100ms），免費額度對個人工作室充裕，Workers + D1 在台灣也有良好的覆蓋。我們更深入的技術分享會在 技術日誌 陸續更新。

我們的原則

  • 誠實優先於漂亮 — 不用行銷話術美化未完成的功能，就說「還在做」。
  • 原創優先於模板 — 寧可做慢做小，也要是自己的作品，不抄。
  • 玩家優先於數據 — 不做會降低遊戲體驗的黑暗模式（強迫登入、逼簽到）。
  • 開源精神 — 非商業核心的工具盡可能開放原始碼或教學筆記。
  • 隱私至上 — 只收集提供服務必要的最少資料，細節見隱私權政策。
