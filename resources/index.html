Resources · Toolbox

工作室日常的工具箱

這是一份「我自己每天用」的工具列表 — 從寫程式、做美術、剪片到接案管理。每一項都標明免費 / 付費，並附短評。沒列上來的就是「試過但已經換掉」的工具。

1. 開發環境

寫程式、版控、終端機、AI pair programmer。

Cursor / VS Code
Mix
編輯器 · GitHub Copilot 替代
Cursor 在 AI 補全速度與整合度上明顯優於 VS Code + Copilot。免費額度夠日常用，重度需求 $20/月。VS Code 還是備用。
Claude Code (CLI)
Paid
AI · 全代理協作
不只是補全，而是能自主跑指令、改檔、debug 的 agent。比 Cursor 適合重構與大型改動。Anthropic 訂閱制。
GitHub
Free
Git host · CI
免費版的 private repo + Actions 對個人專案 100% 夠用。Issues 充當任務管理。
Wrangler CLI
Free
Cloudflare 部署工具
部署 Pages / Workers / D1 一個 CLI 搞定。比手動上 dashboard 快 10 倍。

2. 後端與服務

CDN、伺服器、資料庫、認證。月成本控制在 NT$200 內的祕訣。

Cloudflare Pages
Free
CDN · 靜態站
免費版無流量上限、商業可用、邊緣節點 300+。本站就是用這個跑。
Cloudflare Workers
Free
Serverless API
每天 10 萬請求免費，零冷啟動。整套遊戲後端 250 行搞定。
Cloudflare D1
Free
SQLite on edge
免費 5 GB、自動備份、邊緣複製。資料量不大時的甜蜜點。
Google OAuth
Free
登入服務
完全不用自己管密碼。記得在 worker 端驗 aud 跟 exp，不然帳號接管風險。

3. AI 工具

寫文、生圖、配樂、生程式碼。AI 真的把獨立工作室的產能拉高一個數量級。

Claude (Anthropic)
Paid
寫作 · 程式碼
本工作室的主力 AI。長上下文與複雜推理優於同價位產品。Pro 訂閱 $20/月。
Stable Diffusion (本地)
Free
生圖 · 商業可用
用 Automatic1111 或 ComfyUI 跑在自己 GPU 上。商業條款明確，沒授權問題。
Suno / Udio
Mix
AI 配樂
遊戲 BGM 用，免費版每天 10 首。商業需要付費 plan，但比委託作曲師便宜 20+ 倍。
ElevenLabs
Mix
語音合成
教學影片配音用。中文聲音自然度比 Google TTS 高一個檔次。

4. 設計與美術

UI 設計、向量繪圖、像素編輯、版型。

Figma
Mix
UI / UX 設計
免費版的 3 個 file 限制夠個人用。需要團隊協作或更多檔案才考慮付費。
Aseprite
Paid
像素圖編輯
$20 一次買斷終生用。AI 生圖後的微調修補必備。Steam 也有賣。
Inkscape
Free
向量繪圖
Illustrator 免費替代品。SVG icon、logo 製作首選。
Coolors
Free
調色盤
產生配色方案。本站的藍紫漸變就是用這個試出來的。

5. 影片與動畫

3D 建模、特效、剪片、動畫。

Blender
Free
3D · 動畫
完全免費的 3D 全套。本站 hero 的 3D liquid chrome 就是 Blender 出的。
DaVinci Resolve
Free
剪片 · 調色
免費版功能比付費 Adobe 還多。教學影片剪輯首選。
Houdini Apprentice
Free
VFX · 程序化
免費版有功能限制與浮水印，但學習用 OK。商業需 Indie ($269/年) 或 FX ($1,995/年)。
Maya
Paid
3D 動畫主力
業界標準。Indie 版 $300/年。Blender 用熟後其實可以省掉這個。

6. 商務與生產力

接案、發票、版權、SEO 工具。

Google Search Console
Free
SEO 監控
每個網站都該裝。實時看 Google 怎麼看你的網站。
Cloudflare Web Analytics
Free
隱私友善 analytics
不用 cookie、不存個資、零追蹤。GDPR 不用怕。比 Google Analytics 簡潔。
Gumroad
Mix
數位商品銷售
10% 抽成（含金流）。模板、教學、原始碼販售首選。
藍新金流
Paid
台灣金流
本土客戶用。手續費 2.x%。對於台幣計價、開發票場景比 Stripe 簡單。

7. 我刻意不用的工具

列出來給有需要避雷的朋友。理由都很主觀，僅供參考。

Notion
Mix
為什麼不用：太重
用了 2 年發現 90% 場景純文字編輯器就夠。改用 Obsidian + Markdown，啟動快、檔案歸我管。
Vercel (個人)
Mix
為什麼不用：費用爬升快
Hobby 禁商業使用。一旦有營收就要 $20/月起。同樣需求 Cloudflare 免費可解決。
Adobe 全家桶
Paid
為什麼不用：CC 月費高
$60+/月。對只偶爾用 PS / AI 的場景不划算。用 Photopea + Inkscape + DaVinci 免費組合。
